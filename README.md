# Visão Geral
Este é um projeto simples em PHP/Laravel que exibe uma rota para "Sobre"

Projeto construído para a disciplina **Desenvolvimento de WebServices** do curso **Tecnologia em Sistemas para Internet (TSI) do Senac-SP**.
